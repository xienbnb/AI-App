# 灵犀写作 - AI 写作助手 设计文档

## 产品灵魂与意象锚点

**意象：** 书房里的智能台灯——暖光洒在笔记本上，一支钢笔搁在纸边，窗外是安静的夜色。

**气质关键词：** 专注、灵感、温暖、高效、智能

## 设计风格：柔和卡片风 (Soft Card UI)

选择理由：AI 工具类应用适合科技、高效、通透的视觉语言。柔和卡片的双层阴影和暖灰白底色营造出「纸张质感」，与写作场景天然契合。

## 配色方案

| Token | 颜色 | 意象来源 |
|-------|------|----------|
| 主色 | `#6C63FF` (薰衣草紫) | 灵感与创造力的代表色 |
| 主色渐变 | `#896BFF` | 紫向粉的过渡，温柔有层次 |
| 辅色 | `#FF6584` (珊瑚粉) | 点缀与高亮，如笔尖的红色墨水 |
| 背景 | `#F0F0F3` (暖灰白) | 纸张的温润质感 |
| 文字主 | `#2D3436` | 深灰接近黑色，阅读舒适 |
| 文字次 | `#636E72` | 辅助信息灰度 |
| 成功 | `#00B894` | 保存/完成的肯定感 |

## 导航结构

- **Stack 导航**（无 Tab Bar），适合线性写作流程
- 页面：首页（文章列表）→ 写作页（AI 生成+编辑）→ 详情页（查看文章）

## 核心页面布局

### 首页
- 顶部：大标题「灵犀写作」+ 欢迎语
- 中部：文章列表（横向信息卡片，展示标题、摘要、时间）
- 右下角：FAB 按钮「新建写作」

### 写作页
- 顶部：返回按钮 + 标题「新建写作」
- 输入区：凹陷式输入框（写作主题/要求）
- 生成按钮：渐变主色按钮
- 内容区：AI 流式输出展示区（打字机效果）
- 底部：保存按钮

### 详情页
- 顶部：返回按钮 + 文章标题
- 正文区：完整文章内容展示
- 底部：删除/编辑操作

## 设计禁忌

- ❌ 不使用纯白 `#FFFFFF` 作为卡片背景
- ❌ 不使用 border 描边区分层级
- ❌ 不使用纯黑阴影
- ❌ 不使用 Emoji 作为图标
- ❌ 不使用底部 Tab Bar（本应用为线性流程）
